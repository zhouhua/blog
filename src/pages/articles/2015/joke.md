> javascript 有一个鲜为人知的运算符，叫“趋向于”， 写作 --> 。可以声明一个变量 然后让它趋向于另一个数。
> ```javascript
> var x = 10;
> while (x --> 0) {
>     console.log(x);
> }
> ```
> 就会打印 9,8,7,6,5,4,3,2,1,0 了
